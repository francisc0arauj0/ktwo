# How to configure the ktwo kernel on Windows

### **Install wsl2**

```bash
wsl --install
```


### **Clone repository**

```bash
git clone https://github.com/francisc0arauj0/ktwo/
```

### **Install tools**

- nasm
- gcc
- qemu
- make
- clang-format

### **Setup Bootloader**
Read [boot.md](boot.md)

### **Build**

```bash
make
```

## Build

### **Windows**

Install wsl2

```bash
wsl --install
```


Clone repository

```bash
git clone https://github.com/francisc0arauj0/ktwo/
```

Install tools

- nasm
- gcc
- grub-mkrescue
- grub-pc-bin
- qemu
- make

Build

```bash
make
```
